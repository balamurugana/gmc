Setting the Charts Preferences

You can customize the console and change the way some of the monitoring data is displayed, as described in this section.

Note: The default duration for all the charts is a week.

To set the display of the charts, perform the following:

  1. Choose Edit > Settings

    The Preferences window is displayed.

  2. Choose Gluster > Charts page from the right pane of the Preferences window.

  3. Select the Aggregated CPU Usage chart period from the drop down list.
  4. Select the Aggregated Network Usage chart period from the drop down list.
  5. Select the CPU Usage chart period from the drop down list.
  6. Select the Memory Usage chart period from the drop down list.
  7. Select the Network Usage chart period from the drop down list.
  8. Click Apply.
  9. Click OK to exit the Preferences window.
