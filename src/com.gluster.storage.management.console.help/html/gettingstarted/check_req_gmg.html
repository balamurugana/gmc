Checking Gluster Management Gateway Minimum Requirements

Gluster Management Gateway is installed on a central server using the Linux package (e.g. rpm for CentOS) published by Gluster. Before you install Gluster Management Gateway, you must verify that your environment matches the minimum requirements described in this section.

You must verify that your central server matches the minimum requirements listed below:

  • Memory: 2 GM RAM
  • Processor: 2 (2.4 GHz each) recommended
  • Disk Space: 10 GB

    The following packages must be installed:

  • Java SE runtime version 1.6 – openjdk => 1.6.0.0
  • tomcat5 => 5.5.23
  • wget
