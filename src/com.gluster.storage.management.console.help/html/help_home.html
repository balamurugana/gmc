Gluster Management Console Help Home Page

Gluster Management Console offers an advanced yet simple-to-use graphical user interface (GUI) to manage your clustered storage environment.

This help explains the console interface when working with Gluster Filesystem. It also describes how you can work with the Gluster Management Console to add servers, create volumes, monitor the system, manage logs, download server configurations, and more.

For information about how to install Gluster Filesystem, refer to the Gluster 3.2 Filesystem Installation Guide . For information about accessing the cluster using client systems, Gluster 3.2: Accessing Data - Setting Up GlusterFS Client.
