Create Cluster

You can use Gluster Management Console to create and manage virtual machines and resources on GlusterFS. To create a cluster, perform the following steps.
  1. Open your web browser.
  2. Type the following URL in your web browser:

    3. http://server-name:webserver-port/glustermg/gluster-management-console.jnlp

    Gluster Management Console login window is displayed.

  3. Type the user name and password in the login window and click Login.

    Select Cluster window is displayed.

  4. Select Create. Type the name of the new cluster in the Cluster Name field.
  5. Click OK. The Gluster Management Console window is displayed with the newly created cluster
