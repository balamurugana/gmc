Select Cluster

To select a cluster, perform the following steps.
  1. Open your web browser.
  2. Type the following URL in your web browser:

    3. http://server-name:webserver-port/glustermg/gluster-management-console.jnlp

    Gluster Management Console login window is displayed.

  3. Type the user name and password in the login window and click Login.

    Select Cluster window is displayed.

  4. Select the cluster name from the Cluster Name drop down list.
  5. Click OK.The Progress Information window is displayed.

  6. The Gluster Management Console window displays the selected cluster.
