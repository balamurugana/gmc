Register Cluster

To register a cluster, perform the following steps.
  1. Open your web browser.
  2. Type the following URL in your web browser:

    3. http://server-name:webserver-port/glustermg/gluster-management-console.jnlp in your browser.

    A login window is displayed.

  3. Type the user name and password in the login window and click Login.

    Select Cluster window is displayed.

  4. Select Register. Type the name of the cluster and server name in the respective fields.
  5. Click OK. The Gluster Management Console window is displayed with the newly registered cluster
