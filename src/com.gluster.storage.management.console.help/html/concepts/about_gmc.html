About Gluster Management Console

Gluster Management Console is a Java application that provides a single point of control to build, manage, and maintain your Gluster storage cluster. You can use theconsole's GUI interface to perform the administrative tasks required to create and maintain an integrated environment.

You can use the Gluster Management Console to:

  • Create a Gluster Storage Cluster
  • Expand your cluster
  • Monitor systems in your cluster
  • Manage logs
