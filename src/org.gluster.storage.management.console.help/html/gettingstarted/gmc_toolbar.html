Gluster Management Console Toolbar

The Toolbar provides shortcuts to the main Gluster Management Console functions.

The table below describes the buttons on the toolbar of the console.

Icon Description
Launches the terminal window that allows you type system commands.
Opens the Preferences window.
Refreshes the application.
Allows you to add a server to a cluster.
Allows you to removes a server from a cluster.
Opens the Create Volume window.
Allows you to start a volume.
Allows you to stop a volume.
Allows you to removes a volume.
Allows you to rebalance a volume.
Allows you to reset the volume options.
Allows you to download logs.
Opens the Add Brick window.
Removes brick from a volume.
Migrates a brick from one volume to another.
Allows you to start a task from the Tasks tab.
Allows you to stop a task from the Tasks tab.
Allows you to pause a task from the Tasks tab.
Allows you to clear completed task from the Tasks tab.
